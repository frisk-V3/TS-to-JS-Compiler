# TS-to-JS-Compiler
ブラウザ上で動作する、軽量な **TypeScript から JavaScript へのトランスパイル（変換）ツール** です。
外部の `typescript.js` ライブラリ（unpkg CDN）を利用しており、複雑なビルド環境を構築することなく、ブラウザひとつでTSコードの挙動をテストできます。

##  特徴

- **環境構築不要**: `index.html` をブラウザで開くだけで即座に動作します。
- **リアルタイム変換**: 入力エリアにコードを書くと、即座にJSが出力されます。
- **ダークモードUI**: VS Code風のカラーテーマを採用。
- **最新のTS対応**: CDN経由で常に安定したTypeScriptエンジンを使用します。

##  使い方

1. `index.html` をブラウザで開きます。
2. 左側の「TypeScript (入力)」エリアに、型定義を含むコードを入力します。
3. 右側の「JavaScript (出力)」エリアに、即座にコンパイルされた結果が表示されます。

## 技術仕様

- **Engine**: TypeScript (via unpkg CDN)
- **Target**: ES2015 (ES6)
- **Module**: CommonJS
